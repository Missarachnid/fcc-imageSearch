Image Search Abstraction Layer Project

User Stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. can paginate through the responses by adding a ?offset=2 parameter to the URL
  3. I can get a list of the most recently submitted search strings.

Example urls to retrieve images:

https://corgi.glitch.me/api/imagesearch/corgis
https://corgi.glitch.me/api/imagesearch/siborgis

Example urls to see latest requests:

https://corgi.glitch.me/api/latest/imagesearch

Example latest search return

{"term":"corgi","when":"2018-04-05T17:13:14.742Z"}
